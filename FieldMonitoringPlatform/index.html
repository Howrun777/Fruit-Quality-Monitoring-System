Core Modules
三大技术模块
全链路覆盖
从传感器感知到 Web 端可视化，提供端到端的智慧农业闭环方案。
01

无损糖度检测

AS7341 多光谱传感器采集 11 个波段光谱反射率，配合自研 MLR 多元线性回归模型，实时推算樱桃 Brix 糖度值。无需破损果肉，保留果实完整性，秒级精准出结果，精度达 ±0.3°Brix。

极速 · 高精度 · 无损
02

果园环境监控

SHT31 温湿度 + BH1750 光照传感器全天候采集，建立气候−糖度关联模型，辅助果实成熟期预测与精准灌溉决策。

全天候守护
03

智能监控大屏

ECharts 驱动的 Web 端可视化大屏，支持历史趋势折线图、多维数据分析与阈值告警，多终端自适应布局。

可视化分析
04

安全后台系统

加密登录保护，权限分级管理，所有历史数据持久化至 MySQL 数据库，支持导出与远程运维。

安全 · 可靠
05

断线自动重连

网络容错机制设计，Wi-Fi 中断后自动重试，ESP32 本地缓存数据，确保恶劣环境下数据零丢失。
